# Train CIFAR10 with VGG

## Training
请使用以下代码训练CNN模型          

### 32*32输入 VGG19
python main.py --model VGG19                            

### 224*224输入 添加9层后的VGG19
python main.py --model VGG19_pro

### 224*224输入 删除1层后的VGG11
python main.py --model VGG11_mini
